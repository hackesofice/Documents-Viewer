## Disclaimer
This is an extension / plugin specially designed for the acode app, it adds the capability in acode app to open/view multiple kinds of documents including docx, pdf etc.

## Supported Documents
the plugin supprts multiple kinds of documents hear is the list -
- EXCEL
- DOCX
- CSV
- PDF
- MD
